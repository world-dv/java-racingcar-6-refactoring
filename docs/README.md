# 자동차 경주 

## 기능 구현 목록
- 입력
  - [x] 자동차 이름 리스트 생성(쉼표 구분)
  - [x] 자동차 이름 5자 이하로 제한
- 기능
  - [x] 전진 조건 생성
    - [x] 0에서 9사이에서 무작위 값 생성
  - [x] 전진 기능
    - [x] 무작위 값이 4 이상일 경우 판단
    - [x] 무작위 값이 4 이상일 경우 전진
  - [x] 전진한 만큼 '-' 추가
  - [x] 단독/공동 우승자 판단
- 예외 처리
  - [x] 자동차 이름이 5자리 이하가 아닌 경우
  - [x] 자동차 이름이 쉼표가 아닌 다른 기호로 구분된 경우(특수 문자 포함)

## 입출력 구현 목록
- 입력
  - [x] 자동차 이름 입력
  - [x] 자동차 전진 시도 횟수 입력
- 출력
  - [x] 자동차 이름 입력 문구
  - [x] 자동차 전진 시도 횟수 입력 문구
  - [x] 실행 결과 문구
  - [x] 각 차수별 실행 결과
    - [x] 전진하는 자동차 이름
    - [x] 전진한 횟수
    - [x] 입력 순서대로 자동차 이름과 경주 문구
  - [x] 단독 우승자 안내 문구
  - [x] 공동 우승자 안내 문구
